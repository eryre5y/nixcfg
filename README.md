## Installation

```
# thanks dan4ik605743 for motivation of learning nixos further

nix-shell -p git
cd
git clone https://github.com/eryre5y/nixcfg
cp -r ~/nixcfg/system/* /mnt/etc/nixos/
nixos-install
```
